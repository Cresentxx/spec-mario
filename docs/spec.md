# Spec: Super Mario Bros. — Feature Specification

> 版本: 1.0 | 日期: 2026-05-20 | 状态: Draft

---

## 1. Overview

### 1.1 项目描述

复刻经典 NES 游戏《Super Mario Bros.》(1985) 的核心玩法体验，构建一个功能完整的超级马里奥桌面游戏。玩家操控 Mario 穿越 2 个世界、8 个关卡，体验原版基础机制。

### 1.2 目标用户

- 怀旧游戏玩家
- 平台跳跃游戏爱好者
- 学习游戏开发的学生/开发者

### 1.3 项目范围

- **In Scope**: 单人模式、2 个世界 (地面 + 城堡)、8 个关卡、完整游戏循环 (标题→游戏→通关/Game Over)、所有基础敌人、道具、方块系统、音效
- **Out of Scope**: 双人模式、水下关卡、地下关卡、关卡编辑器、联机功能

---

## 2. User Stories

### US-1: 玩家移动与跳跃

**作为玩家**，我希望能够精确控制 Mario 的移动和跳跃，以便在各种地形中穿行。

**验收标准**:
- [ ] Mario 可以左右移动，速度由加速度逐渐增加
- [ ] 按住跳跃键时间越长跳得越高 (可变跳跃高度)
- [ ] 按住加速键可跑步，跑步时跳跃距离更远
- [ ] 空中可调整水平方向
- [ ] 高速时反方向按键产生刹车效果
- [ ] 操作手感接近原版 (物理参数见 PRD §3.2)

---

### US-2: 玩家状态转换

**作为玩家**，我希望通过拾取道具改变 Mario 的状态和能力，以应对不同的挑战。

**验收标准**:
- [ ] Small Mario: 1 格高，基础移动/跳跃，碰到敌人即死
- [ ] Super Mario: 2 格高，可顶碎砖块，受击退回 Small
- [ ] Fire Mario: 2 格高，可发射火球 (最多 2 发同屏)，受击退回 Super
- [ ] 状态转换有视觉变化 (外观大小/颜色变化)
- [ ] 拾取 Starman 后进入 10 秒无敌状态，接触敌人直接击杀
- [ ] 无敌期间角色闪烁发光

---

### US-3: 方块互动

**作为玩家**，我希望从下方顶击方块获取道具或开辟通路，以增强游戏策略性。

**验收标准**:
- [ ] 问号方块 (?) 顶击后掉落道具并变为空方块
- [ ] 砖块在 Super/Fire 状态下顶击碎裂，Small 状态仅弹跳
- [ ] 隐藏方块顶击后出现 (通常含绿蘑菇或金币)
- [ ] 已激活方块变为实心 Used Block
- [ ] 方块被顶时有弹跳动画
- [ ] 砖块碎裂有碎片飞散动画
- [ ] 道具从方块弹出有出现动画

---

### US-4: 道具拾取

**作为玩家**，我希望拾取散布在关卡中的道具获得增益效果。

**验收标准**:
- [ ] Super Mushroom (红蘑菇): Small → Super；已是 Super 则 +1000 分
- [ ] Fire Flower (火球花): 任意状态 → Fire；Small 时先变大
- [ ] Starman (无敌星): 10 秒无敌
- [ ] 1-UP Mushroom (绿蘑菇): 生命 +1
- [ ] Coin (金币): 每 100 枚 = +1 生命，每枚 +200 分
- [ ] 蘑菇弹出后向右移动，遇壁反弹
- [ ] 火球花直接在方块上方出现
- [ ] 无敌星弹出后弹跳移动

---

### US-5: 地面敌人遭遇

**作为玩家**，我希望与地面敌人进行有策略的战斗，通过踩踏或火球击杀它们。

**验收标准**:
- [ ] Goomba: 直线行走，踩踏后扁平消失
- [ ] Koopa Troopa (绿): 直线行走遇壁转向，踩踏后缩壳可踢飞
- [ ] Koopa Troopa (红): 巡逻不落崖，踩踏后缩壳可踢飞
- [ ] Buzzy Beetle: 直线行走，免疫火球，踩踏后缩壳
- [ ] Hammer Bro: 跳跃巡逻，投掷锤子，踩踏或两发火球击杀
- [ ] 飞行龟壳击杀沿途敌人，连杀计分递增
- [ ] 飞行龟壳碰墙反弹，可伤害玩家

---

### US-6: 特殊敌人遭遇

**作为玩家**，我希望在特定场景中遇到不同类型的敌人，增加游戏多样性。

**验收标准**:
- [ ] Piranha Plant: 从管道中周期性伸缩，火球/无敌可击杀
- [ ] Bullet Bill: 从画面边缘直线飞入，踩踏/火球击杀
- [ ] Lakitu: 云上漂浮投掷 Spiny，踩踏/火球击杀
- [ ] Spiny: 直线行走，不可踩踏，仅火球击杀
- [ ] Podoboo: 岩浆中周期性弹跳 (城堡关卡)，仅无敌可击杀

---

### US-7: 地面关卡体验 (World 1)

**作为玩家**，我希望体验 4 个难度递增的地面主题关卡，学习基础技能后逐步挑战。

**验收标准**:
- [ ] World 1-1: 标准关卡，入门难度，教授基本操作
- [ ] World 1-2: 进阶关卡，引入更多敌人类型和地形变化
- [ ] World 1-3: 挑战关卡，高难度，需要精确跳跃
- [ ] World 1-4: Boss 关卡，对抗 Bowser (踩断桥梁/火球击杀)
- [ ] 每关有旗杆通关流程 (下滑动画 + 高度得分)
- [ ] 管道可进入隐藏金币房间
- [ ] 不可回退的摄像机滚动
- [ ] 每关 400 秒时间限制

---

### US-8: 城堡关卡体验 (World 2)

**作为玩家**，我希望体验 4 个城堡主题关卡，面对火焰陷阱和最终 Boss。

**验收标准**:
- [ ] World 2-1: 城堡标准关卡，引入火焰棒和岩浆
- [ ] World 2-2: 进阶关卡，更多火焰陷阱和移动平台
- [ ] World 2-3: 挑战关卡，高密度火焰棒和敌人组合
- [ ] World 2-4: 最终 Boss 关卡，Bowser 来回跳跃喷火投掷锤子
- [ ] 城堡背景为黑色，灰色方块，岩浆和火焰视觉
- [ ] 掉入岩浆即死
- [ ] 击败 World 2-4 Bowser 后游戏通关

---

### US-9: 计分与生命管理

**作为玩家**，我希望有明确的计分和生命系统，激励我提高表现。

**验收标准**:
- [ ] 踩踏敌人 +100，火球击杀 +200，踢壳连杀递增 (200→400→...→1UP)
- [ ] 拾取金币 +200，蘑菇/花/星 +1000
- [ ] 旗杆高度得分 (最高 5000，中高 2000-4000，中低 100-800)
- [ ] 通关后剩余时间 × 50 计入总分
- [ ] 初始 3 条生命
- [ ] 100 枚金币 = +1 生命
- [ ] 拾取 1-UP 蘑菇 +1 生命
- [ ] 踢壳连杀第 8 个敌人 +1 生命
- [ ] 生命耗尽 → Game Over 画面

---

### US-10: 游戏流程与状态

**作为玩家**，我希望有完整的游戏流程，从标题画面到通关或 Game Over。

**验收标准**:
- [ ] 标题画面: Logo + "START GAME" 选项 + 背景动画
- [ ] 开始游戏后进入 World 1-1
- [ ] 通关旗杆流程: 接触旗杆 → 下滑动画 → 走入城堡 → 计分 → 下一关
- [ ] 死亡: 掉坑/碰敌(Small)/岩浆/时间耗尽 → 生命 -1 → 重试当前关
- [ ] Game Over: 生命为 0 → Game Over 画面 → 可选 Continue (从当前 World 起点重新开始)
- [ ] 通关: 击败 World 2-4 Bowser → 结局动画 → 可选 Second Quest
- [ ] 暂停: Enter/P 键暂停，画面静止显示 "PAUSED"
- [ ] HUD 实时显示: 分数、金币、关卡编号、剩余时间

---

### US-11: 音效与音乐

**作为玩家**，我希望听到贴合场景的背景音乐和音效，增强沉浸感。

**验收标准**:
- [ ] 地面关卡播放 Overworld Theme
- [ ] 城堡关卡播放 Castle Theme
- [ ] 无敌状态切换为 Starman Theme
- [ ] 通关、Game Over、标题画面各有专属音乐
- [ ] 跳跃、金币、顶方块、砖块碎裂、道具拾取、火球发射、踩踏敌人、受伤、死亡、1-UP、旗杆下滑、管道进出、暂停均有音效
- [ ] 暂停时背景音乐停止

---

### US-12: 画面特效

**作为玩家**，我希望看到丰富的视觉反馈，使游戏体验更生动。

**验收标准**:
- [ ] 金币从方块弹出旋转上升
- [ ] 砖块碎裂 4 片碎片飞散
- [ ] Goomba 被踩后扁平动画
- [ ] 击杀/拾取时分数数字上浮消失
- [ ] 无敌状态角色闪烁
- [ ] 旗杆条件满足时放烟花
- [ ] Mario 死亡上浮后下落消失

## 3. Non-Functional Requirements

### 3.1 性能

- 目标帧率: 稳定 60 FPS
- 启动时间: < 3 秒
- 关卡切换: < 1 秒
- 内存占用: < 150 MB
- 输入延迟: < 16ms (1 帧)

### 3.2 兼容性

- 操作系统: Windows 10 (21H2+) / Windows 11
- 架构: x64
- 运行方式: 单一 .exe 可执行文件，无需安装 Python

### 3.3 打包

- 使用 PyInstaller 或 Nuitka 打包为 .exe
- 最终体积 < 50 MB
